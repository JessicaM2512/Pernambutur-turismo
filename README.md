# Pernambutur Turismo

Site de turismo com passeios e transfers saindo de Recife, Pernambuco.

## Tecnologias

- Vite
- TypeScript
- React
- Tailwind CSS

## Como executar

```
sh
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```

## Deploy

Para fazer deploy, você pode usar serviços como Vercel, Netlify ou GitHub Pages.
